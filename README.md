<h1>DoNote</h1>
<p>โปรเจกต์นี้เป็นระบบจัดการงานแบบกระดาน (board-based task management) ชื่อ DoNote โดย Frontend ใช้ Next.js
แบบ App Router และ Backend ใช้ Hono รันบน Cloudflare Workers ฟีเจอร์สําคัญคือ สมัครสมาชิกด้วย OTP, ล็อกอิน, สร้างบอร์ด, แชร์บอร์ดด้วย invite code, 
เพิ่มสมาชิก, สร้างงาน, assign งาน, แจ้งเตือนผู้เกี่ยวข้อง, รวมงานของตัวเองและมุมมองปฏิทินตาม due date</p>
